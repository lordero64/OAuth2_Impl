# 🔐 OAuth2 GitHub Authentication Demo

Учебный проект с аутентификацией через GitHub OAuth2 и разделением ролей USER/ADMIN.

## 🎯 О проекте

Учебный проект, демонстрирующий интеграцию Spring Security с OAuth2 провайдером GitHub. Приложение позволяет пользователям входить через GitHub, назначает роли (USER/ADMIN) и разграничивает доступ к защищенным ресурсам.

**Основные возможности:**
- ✅ OAuth2 аутентификация через GitHub
- ✅ Автоматическое назначение ролей на основе GitHub login
- ✅ Разграничение доступа для USER и ADMIN
- ✅ REST API для получения данных пользователя
- ✅ Защита эндпоинтов с помощью Spring Security
- ✅ Обработка ошибок (401, 403)
- ✅ Логирование ключевых событий
- ✅ Безопасный выход из системы

---

## ⚙️ Функциональность

| Функция | Описание |
|---------|----------|
| **Вход через GitHub** | Пользователь авторизуется через GitHub OAuth2 |
| **Профиль пользователя** | Отображение данных: логин, имя, аватар, роль |
| **Роли** | USER (по умолчанию) и ADMIN (по списку логинов) |
| **Админ-панель** | Доступна только пользователям с ролью ADMIN |
| **REST API** | JSON эндпоинты для получения данных |
| **Выход** | Очистка сессии и отзыв токена через GitHub |
